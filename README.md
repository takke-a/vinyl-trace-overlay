# Vinyl Trace Overlay

Forza Horizon 6 のバイナルグループ作成を補助する、参照画像オーバーレイツールです。  
ゲーム画面の上に半透明の参照画像を重ねて表示することで、バイナルを組み合わせてキャラクターをトレース制作できます。

---

## 動作環境

- **OS**: Windows 10 / 11
- **Python**: 3.10 以上（スクリプト版のみ必要）
- **依存ライブラリ**: Pillow（自動インストール）

---

## 起動方法

### 方法 1 — スクリプトで起動（Python 必要）

```
run.bat をダブルクリック
```

初回起動時に Pillow が自動インストールされます。

### 方法 2 — EXE で起動（Python 不要）

```
build_exe.bat をダブルクリック → dist/VinylTraceOverlay.exe が生成されます
```

生成された `VinylTraceOverlay.exe` は単体で動作します。

---

## 基本的な使い方

1. ゲームを **ウィンドウモード（ボーダーレス推奨）** で起動
2. `Vinyl Trace Overlay` を起動し、**Open Image** で参照画像を開く
3. 透過率・サイズを調整してゲーム画面の上に配置
4. **F2** でクリック透過をONにすると、オーバーレイを表示したままゲームを操作できる
5. バイナルエディタ上でトレースしながら図形を配置

## 機能一覧

| 機能 | 説明 |
|------|------|
| **Open Image** | PNG / JPG / BMP / WebP などの画像を読み込む |
| **Opacity** | ウィンドウ全体の透過率（10〜100%） |
| **Scale** | 画像の表示サイズ（5〜400%） |
| **Fit** | 画像をキャンバスに合わせてリサイズ |
| **1:1** | 倍率を100%にリセット |
| **Reset View** | パン（移動）をリセットして中央に戻す |
| **Flip H / V** | 水平・垂直反転（左右対称キャラに便利） |
| **Grid** | センター基準のグリッド線を表示 |
| **Click-Thru** | クリックがオーバーレイを透過してゲームに届く（Windows専用） |
| **Lock Pos** | ウィンドウ位置を固定して誤移動を防止 |
| **Grid Color** | グリッド線の色を自由に変更 |
| **Cell Size** | グリッドのマス目サイズ（px）を変更 |

### 表示モード

| モード | 用途 |
|--------|------|
| Normal | 通常表示 |
| Edge Detect | 輪郭線のみ強調（トレースライン確認に便利） |
| Grayscale | グレースケール表示 |
| Invert | 色反転 |
| High Contrast | コントラスト強調（細部確認に） |
| Soft Glow | ソフト発光エフェクト |

### ホバー情報

画像の上にマウスを乗せると、ステータスバーにそのピクセルの  
`座標 / RGB値 / HEXカラーコード` がリアルタイム表示されます。  
ゲーム内でカラーを合わせる際にご活用ください。

## キーボードショートカット

| キー | 機能 |
|------|------|
| `Ctrl + O` | 画像を開く |
| `Scroll` | ズームイン / ズームアウト |
| `右クリック + ドラッグ` | 画像をパン（移動） |
| `F1` | コントロールパネル 表示/非表示 |
| `F2` | クリック透過 ON/OFF |
| `F3` | グリッド表示 ON/OFF |
| `F11` | 全画面モード切替 |
| `Ctrl + =` / `Ctrl + -` | ズームイン / ズームアウト |
| `Escape` | 全画面中は全画面解除 / 通常時は終了 |

## EXE ビルドに関するトラブルシューティング

| 症状 | 対処 |
|------|------|
| `python not found` エラー | Python を公式サイトからインストール。インストール時に「Add to PATH」にチェック |
| ビルドは完了するが EXE が起動しない | アンチウイルスソフトの除外設定に `dist` フォルダを追加 |
| `pip install` が失敗する | コマンドプロンプトを**管理者として実行**して `build_exe.bat` を起動 |
| ビルド中に止まる | `build_exe.bat` をコマンドプロンプトから直接実行してエラーを確認 |

## ワークフロー例（キャラクタートレース）

```
1. キャラクターの正面・側面の参照画像を用意する
2. FH6 のバイナルエディタを開く（ウィンドウモード）
3. Vinyl Trace Overlay を起動 → 参照画像を開く
4. Mode: Edge Detect でアウトラインを確認
5. Grid をONにして体の比率を把握
6. Scale と Opacity を調整してゲーム画面に重ねる
7. F2（クリック透過）をONにしてバイナル配置開始
8. 顔パーツ（肌・目・眉・口・髪）を別レイヤーに分けて制作
9. 完成後、F2をOFFにしてオーバーレイを確認
```
