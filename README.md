# Vinyl Trace Overlay

Forza Horizon 6 のバイナルグループ作成を補助する、参照画像オーバーレイツールです。  
ゲーム画面の上に半透明の参照画像を重ねて表示することで、バイナルを組み合わせてキャラクターをトレース制作できます。

## ダウンロード

> **[Releases ページから VinylTraceOverlay.exe をダウンロード](https://github.com/takke-a/vinyl-trace-overlay/releases/latest)**

※ このリポジトリをZIPでダウンロードしてもEXEは含まれません。上のリンクからダウンロードしてください。

---

## 動作環境

- **OS**: Windows 10 / 11
- **Python**: 3.10 以上（スクリプト版のみ必要）
- **依存ライブラリ**: Pillow（自動インストール）

---

## 起動方法

### 方法 1 — EXE で起動（推奨・Python 不要）

`VinylTraceOverlay.exe` をダブルクリックするだけで起動します。

### 方法 2 — スクリプトで起動（Python 必要）

```
run.bat をダブルクリック
```

初回起動時に Pillow が自動インストールされます。

## 基本的な使い方

1. ゲームを **ウィンドウモード（ボーダーレス推奨）** で起動
2. `Vinyl Trace Overlay` を起動し、**Open Image** で参照画像を開く
3. 透過率・サイズを調整し、画像をドラッグしてゲーム画面上に配置
4. **Click-Thru** を長押し（約0.5秒）でONにするとゲームを操作できる
5. **N キー長押し**でオーバーレイを一時的に非表示にしてForzaの出来を確認
6. バイナルエディタ上でトレースしながら図形を配置

## 機能一覧

| 機能 | 説明 |
|------|------|
| **Open Image** | PNG / JPG / BMP / WebP などの画像を読み込む |
| **Opacity** | ウィンドウ全体の透過率（10〜100%） |
| **Scale** | 画像の表示サイズ（5〜400%） |
| **Fit** | 画像をキャンバスに合わせてリサイズ |
| **1:1** | 倍率を100%にリセット |
| **Reset View** | パン（移動）をリセットして中央に戻す |
| **Flip H / V** | 水平・垂直反転（左右対称キャラに便利） |
| **Grid** | センター基準のFH6スタイルグリッドを表示 |
| **Click-Thru** | クリックがオーバーレイを透過してゲームに届く（Windows専用） |
| **Lock Pos** | ウィンドウ位置を固定して誤移動を防止 |
| **Light BG** | 背景をFH6のバイナルエディタに近い明るい色にする |
| **Grid Color** | グリッド線の色を自由に変更 |
| **Grid Size** | グリッドのマス目サイズ（px）を変更 |
| **Layers** | ゲーム内のレイヤー数（0〜3000）を追跡するカウンター |
| **FH6 HSBサンプラー** | ホバー中のピクセルをFH6形式（0.00〜1.00）でHSB/RGB表示・コピー |

### 表示モード

| モード | 用途 |
|--------|------|
| Normal | 通常表示 |
| Edge Detect | 輪郭線のみ強調（トレースライン確認に便利） |
| Grayscale | グレースケール表示 |
| Invert | 色反転 |
| High Contrast | コントラスト強調（細部確認に） |
| Soft Glow | ソフト発光エフェクト |

### ホバー情報

画像の上にマウスを乗せると、ステータスバーにそのピクセルの  
`座標 / RGB値 / HEXカラーコード / HSB値（0.00〜1.00）` がリアルタイム表示されます。  
HSBパネルの **Copy** をクリックするとFH6のカラーピッカーにそのまま貼り付けられる形式でコピーされます。

## キーボードショートカット

| キー | 機能 |
|------|------|
| `Ctrl + O` | 画像を開く |
| `Scroll` | ズームイン / ズームアウト |
| `左クリック + ドラッグ` | 画像を移動 |
| `右クリック + ドラッグ` | 画像をパン（移動） |
| `F1` | コントロールパネル 表示/非表示 |
| `F2`（長押し約0.5秒） | クリック透過 ON/OFF（フォーカスがなくても動作） |
| `F3` | グリッド表示 ON/OFF |
| `F4` | Light BG ON/OFF |
| `F5` | HSBカラーロック ON/OFF |
| `N`（長押し） | Peek（押している間だけ非表示・Forzaが前面に出る） |
| `Ctrl + C` | HSBカラーをコピー |
| `Ctrl + =` / `Ctrl + -` | ズームイン / ズームアウト |
| `Ctrl + ↑` / `Ctrl + ↓` | レイヤーカウンター +1 / -1 |
| `F11` | 全画面モード切替 |
| `Escape` | 全画面中は全画面解除 / 通常時は終了 |

> すべてのショートカットは **Settings タブ** から自由に変更できます。

## コントロールパネルのリサイズ

パネルとキャンバスの境界（細い区切り線）を上下にドラッグすると、パネルの高さを自由に変更できます。  
高さに応じて5段階でレイアウトが自動切り替えされるため、どの高さでも全機能にアクセスできます。

| パネル高さ | レイアウト |
|-----------|-----------|
| 最小 | 2行コンパクト |
| ↓ | 2行ゆったり |
| ↓ | 3行 |
| ↓ | 5行 |
| 最大 | 6行フル |

## EXE ビルドに関するトラブルシューティング

| 症状 | 対処 |
|------|------|
| `python not found` エラー | Python を公式サイトからインストール。インストール時に「Add to PATH」にチェック |
| ビルドは完了するが EXE が起動しない | アンチウイルスソフトの除外設定に `dist` フォルダを追加 |
| `pip install` が失敗する | コマンドプロンプトを**管理者として実行**して `build_exe.bat` を起動 |
| ビルド中に止まる | `build_exe.bat` をコマンドプロンプトから直接実行してエラーを確認 |

## ワークフロー例（キャラクタートレース）

```
1. キャラクターの正面・側面の参照画像を用意する
2. FH6 のバイナルエディタを開く（ウィンドウモード）
3. Vinyl Trace Overlay を起動 → 参照画像を開く
4. Mode: Edge Detect でアウトラインを確認
5. Grid をONにして体の比率を把握
6. Scale と Opacity を調整してゲーム画面に重ねる
7. Click-Thru（長押し）をONにしてバイナル配置開始
8. N キー長押しでオーバーレイを一時非表示にしてForzaの出来を確認
9. 顔パーツ（肌・目・眉・口・髪）を別レイヤーに分けて制作
10. Layers カウンターでゲーム内の残りレイヤー数を管理
11. 完成後、Click-Thru（長押し）をOFFにしてオーバーレイを確認
```
